Template Installation

It’s easy to install Template . Just follow these steps, they won’t take much of your time.

Install NodeJs

Your computer must have installed nodejs to run this template. You can download Node.js from https://nodejs.org/en. NPM comes bundled with Node.js

Recommended NodeJs version 14.17.4 LTS

Template Installation

Install Yarn

Your computer must have installed Yarn package manager to run this template. You can easily download Yarn from there Github repositories

Go for download

Template Installation

Install Npm Packages

You must have to install packages. You can do this by running yarn install from the root of your project to install all the necessary dependencies.

Template Installation

Development Server

Run yarn start from the root of your project to start a dev server. Navigate to http://localhost:3000/. The app will automatically reload if you change any of the source files.

Template Installation

Build

Run yarn build to build the project. The build artifacts will be stored in the build directory.
